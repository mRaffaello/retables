# Custom renderers

## Table of contents

You can specify a custom renderer for every cell of your table.
In particular:

-   Default renderer for [**all header cells**](#header-cell-default-renderer)
-   Default renderer for [**all body table cells**](#table-cell-default-renderer)
-   Custom renderer for a [**specific header**](#header-cell-custom-renderer)
-   Custom renderer for a [**specific column**](#table-cell-custom-renderer)

In case both a custom renderer and a default renderer are specified, only the custom renderer will be used.

:::info

In the next sections other specific renderer types will be defined for some functionality (pagination, selection ...).

:::
