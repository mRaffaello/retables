# Basic personalization

## Table of contents

In this page you will learn how to:

-   Apply custom classes to the [**table**, **headers** and **rows**](#basic-example)
-   Apply even [**spacing**](#default-cell-spacing)
-   Edit the [**grid appereance**](#edit-the-grid-appereance)

:::info

As referenced before, please keep in mind that, for semplicity, all the examples use [Tailwind](https://tailwindcss.com/) as css framework. However, **reTables** does not restrict you in any way when choosing how to handle **css**.

:::
